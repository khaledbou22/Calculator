# Calculator 

A clean and simple web-based calculator built with HTML, CSS, and JavaScript.

## Live Demo

👉 [Try it here](https://calculator-aqpu.vercel.app/)

## Preview

> A lightweight calculator that runs entirely in the browser — no installations, no dependencies.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clean and minimal UI
- Responsive design — works on both desktop and mobile
- Instant calculations with no page reload

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

If you want to run it locally:

\```bash
git clone https://github.com/khaledbou22/Calculator.git
cd Calculator
\```

Then just open `index.html` in your browser — that's it.
